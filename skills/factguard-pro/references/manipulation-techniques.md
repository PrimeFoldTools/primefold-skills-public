# Manipulation Techniques Database

Understanding common manipulation patterns across framing, emotions, logic, structure, and social contexts.

---

## Core Techniques

### Russell Conjugations
Same action, different framing: "I'm confident / You're arrogant / They're delusional"

### Euphemism Treadmill  
Soft language obscures reality: "Rightsizing" = Layoffs, "Enhanced interrogation" = Torture

### Agency Hiding
Passive voice conceals responsibility: "Mistakes were made" (Who made them?)

### Fear Appeals
Manufactured urgency: "If you don't X, terrible Y will happen"

### Logical Fallacies
Whataboutism, false balance, slippery slope, ad hominem, false dichotomy

### Gish Gallop
Overwhelming with weak arguments: 10+ claims rapidly, too many to address

### Motte-and-Bailey
Defend weak claim by retreating to strong claim, then return to weak

### Moving Goalposts
Change success criteria when met: "Show me evidence" → [shown] → "That's not enough"

---

**Remember:** Verification and critical thinking defeat manipulation. When multiple patterns appear, increase scrutiny.
