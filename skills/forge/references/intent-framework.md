# Intent Framework

Every piece of content must have ONE primary intent to be effective.

---

## Core Principle: One Intent Per Piece

Multiple intents create conflicting optimization targets. Solution: Identify primary, make secondary elements micro.

---

## The Five Intents

### 1. DISCOVERY
Reach new audiences who don't know you exist.
- **Priority:** Algorithmic reach > relationship
- **Platforms:** TikTok, Instagram Reels, YouTube Shorts, Pinterest, X
- **Hook:** Pattern interrupt (1-3 seconds)
- **CTA:** None or very soft
- **Success Metrics:** Reach, viral coefficient, completion rate

### 2. TRUST
Build credibility and relationship.
- **Priority:** Relationship depth > reach
- **Platforms:** LinkedIn, YouTube, Instagram Stories, Email, X
- **Hook:** Credibility signal + value promise
- **CTA:** Relationship deepening
- **Success Metrics:** Engagement depth, save rate, return visitor rate

### 3. CONVERSION
Drive a specific action (sale, signup, booking).
- **Priority:** Action taken > everything else
- **Platforms:** LinkedIn, YouTube, Email, X, Instagram
- **Hook:** Problem agitation or transformation
- **CTA:** ONE specific, clear action
- **Success Metrics:** Conversion rate, CPA, ROI

### 4. RETENTION
Encourage return engagement and ongoing relationship.
- **Priority:** Repeat behavior > one-time action
- **Platforms:** YouTube, Instagram Stories, TikTok, Email, Podcast
- **Hook:** Series setup or callback
- **CTA:** Return behavior
- **Success Metrics:** Return rate, subscription growth, completion rate

### 5. AUTHORITY
Establish expertise and peer recognition.
- **Priority:** Respect from peers > mass appeal
- **Platforms:** LinkedIn, X, YouTube, Medium, Conference speaking
- **Hook:** Contrarian insight or original research
- **CTA:** Minimal or professional connection
- **Success Metrics:** Peer engagement, speaking invites, citations

---

## Intent Sequencing (Funnel Thinking)

Common path:
1. Discovery (reach them)
2. Trust (build relationship)
3. Conversion (drive action)
4. Retention (keep engaged)
5. Authority (peer recognition)

---

**Remember:** One intent per piece. When in doubt, ask: "If this content only achieved ONE thing, what would make it successful?"
