# Repurposing Workflows

Cross-platform adaptation strategies maintaining platform-native feel.

---

## Universal Rules

### Rule 1: Never Mechanical Copy-Paste
Platforms penalize non-native content. Solution: Adapt, don't duplicate

### Rule 2: Aspect Ratio is Non-Negotiable
- Vertical (9:16): TikTok, Instagram Reels, YouTube Shorts
- Square (1:1): Instagram Feed, LinkedIn
- Horizontal (16:9): YouTube, LinkedIn video

### Rule 3: Hook Must Be Platform-Adapted
Different hook execution for same content idea

### Rule 4: Captions/Titles Are Platform-Specific
LinkedIn professional ≠ Instagram casual ≠ TikTok brief

### Rule 5: Success Metrics Change Per Platform
TikTok (completion) ≠ LinkedIn (comment depth) ≠ YouTube (watch time)

---

## Direction Patterns

### Long-Form → Short-Form (Common)
YouTube (10-30min) → TikTok/Reels/Shorts
- Identify clip-worthy moments
- Create new hook
- Add platform-native elements
- Post natively

### Short-Form → Long-Form (High Leverage)
Viral TikTok/Reel → YouTube Explainer
- Analyze why it went viral
- Expand with depth and examples
- Search-optimize title
- Structure for watch time

### Horizontal Moves
TikTok ↔ Instagram Reels, YouTube ↔ LinkedIn, X ↔ LinkedIn

---

## Quality Control Checklist

Before publishing:
- [ ] Aspect ratio correct
- [ ] Hook rewritten (not copied)
- [ ] Caption platform-appropriate
- [ ] Platform-native elements added
- [ ] No watermarks from other platforms
- [ ] Pacing appropriate
- [ ] CTA platform-appropriate
- [ ] Posted natively

---

**Remember:** Repurposing is adaptation, not duplication. Each platform deserves native content.
